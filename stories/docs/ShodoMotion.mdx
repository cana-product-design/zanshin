## The brush stroke as animation model

Shodo — Japanese calligraphy — offers a precise vocabulary for motion that digital animation theory has never named cleanly.

A master calligrapher begins each stroke in *seme* (攻め): the brush approaches the paper with controlled pressure, not yet touching. There is intention before contact. Then comes the strike — *utte* — the decisive moment when ink meets washi. The stroke travels. And then *zanshin* (残心): the brush lifts, but the hand does not pull away. It hovers. The awareness of the completed mark lingers.

This three-phase arc — **seme → utte → zanshin** — is the structural model behind every motion token in this system.

The animation behaviors below are calibrated for this specific ground: deep enough that even a faint asagi line reads, still enough that a single entrance draws the eye.

## The four motion arcs

### 1 — Hiki-fude (引き筆) — The drawn line

The brush pulls across the paper in a single confident stroke. Used for elements that **appear from a direction**: panels sliding in, toasts entering from an edge, navigation expanding.

```css
/* Hiki-fude: directional pull entrance */
@keyframes hiki-fude {
  from {
    opacity: 0;
    transform: translateX(-16px);
  }
  to {
    opacity: 1;
    transform: translateX(0);
  }
}

.hiki-fude {
  animation: hiki-fude var(--duration-long) var(--ease-kendo) both;
}
```

**Easing:** `--ease-kendo` `cubic-bezier(0.25, 0.46, 0.45, 0.94)` — controlled deceleration, like a brush slowing as it nears the end of a stroke.

---

### 2 — Tate-fude (縦筆) — The vertical drop

The brush descends. Ink falls from the tip down the washi. Used for elements that **enter from above**: dropdown menus, modal sheets, notification banners.

```css
/* Tate-fude: vertical descent */
@keyframes tate-fude {
  from {
    opacity: 0;
    transform: translateY(-12px) scaleY(0.96);
    transform-origin: top center;
  }
  to {
    opacity: 1;
    transform: translateY(0) scaleY(1);
  }
}

.tate-fude {
  animation: tate-fude var(--duration-medium) var(--ease-strike) both;
}
```

**Easing:** `--ease-strike` `cubic-bezier(0.0, 0.0, 0.2, 1)` — fast entry with strong deceleration, like the initial pressure of a vertical stroke.

---

### 3 — Fude-okoshi (筆起こし) — The brush awakening

The brush is placed on the paper and pressed before moving. The mark begins as a dot that blossoms outward. Used for elements that **reveal in place**: content blocks fading and rising on scroll, cards appearing on load.

```css
/* Fude-okoshi: reveal in place */
@keyframes fude-okoshi {
  from {
    opacity: 0;
    transform: translateY(8px) scale(0.99);
  }
  to {
    opacity: 1;
    transform: translateY(0) scale(1);
  }
}

.fude-okoshi {
  animation: fude-okoshi var(--duration-long) var(--ease-zen) both;
}
```

**Easing:** `--ease-zen` `cubic-bezier(0.4, 0.0, 0.2, 1)` — balanced entry. Neither urgent nor slow. The mark appears as if it was always there, gradually becoming visible.

---

### 4 — Zanshin-hiki (残心引き) — The lingering exit

When the brush lifts, it does not vanish. It withdraws slowly, leaving the awareness of its presence. Used for all **exits**: closing modals, dismissed toasts, collapsing panels.

```css
/* Zanshin-hiki: the lingering exit */
@keyframes zanshin-hiki {
  from {
    opacity: 1;
    transform: translateY(0) scale(1);
  }
  to {
    opacity: 0;
    transform: translateY(-6px) scale(0.995);
  }
}

.zanshin-hiki {
  animation: zanshin-hiki var(--duration-slow) var(--ease-zanshin) both;
}
```

**Easing:** `--ease-zanshin` `cubic-bezier(0.0, 0.0, 0.1, 1)` — extreme deceleration at the end. The element nearly stops before it fully disappears. The viewer watches it go rather than having it taken away.

## Stagger — the composed composition

A shodo composition is not a single stroke. It is a sequence — each mark placed with awareness of what came before. In interface terms: **staggered reveals** where each element inherits a delay proportional to its position.

```css
/* Staggered scroll reveal — 80ms ma interval between each mark */
[data-shodo-stagger] > * {
  animation: fude-okoshi var(--duration-long) var(--ease-zen) both;
}

[data-shodo-stagger] > *:nth-child(1) { animation-delay: calc(var(--duration-ma) * 0); }
[data-shodo-stagger] > *:nth-child(2) { animation-delay: calc(var(--duration-ma) * 1); }
[data-shodo-stagger] > *:nth-child(3) { animation-delay: calc(var(--duration-ma) * 2); }
[data-shodo-stagger] > *:nth-child(4) { animation-delay: calc(var(--duration-ma) * 3); }
[data-shodo-stagger] > *:nth-child(5) { animation-delay: calc(var(--duration-ma) * 4); }
/* ... extend as needed */
```

The `--duration-ma` token (80ms) is the *ma* interval — the charged pause between Shinto ritual actions. Small enough to read as a single composition, large enough to perceive as intentional sequence.

## Token reference

| Token | Value | Shodo role |
|---|---|---|
| `--ease-kendo` | `cubic-bezier(0.25, 0.46, 0.45, 0.94)` | Seme — controlled approach |
| `--ease-strike` | `cubic-bezier(0.0, 0.0, 0.2, 1)` | Utte — decisive contact |
| `--ease-zen` | `cubic-bezier(0.4, 0.0, 0.2, 1)` | Fude-okoshi — balanced reveal |
| `--ease-zanshin` | `cubic-bezier(0.0, 0.0, 0.1, 1)` | Lingering exit, slow release |
| `--ease-ma` | `cubic-bezier(0.4, 0.0, 0.6, 1)` | Symmetric breath, in/out |
| `--duration-ma` | `80ms` | Ma interval — stagger unit |
| `--duration-short` | `300ms` | Minimum breath — all transitions |
| `--duration-medium` | `500ms` | Standard — dropdowns, badges |
| `--duration-long` | `700ms` | Entrance — section reveals |
| `--duration-slow` | `1000ms` | Zanshin hold — hero, exit |

---

## Rules

1. **No transition faster than 300ms.** `--duration-instant` (150ms) is reserved for kendo snap micro-interactions only — icon swaps, checkbox ticks.
2. **Exits use `--ease-zanshin`.** Every dismissal lingers. Nothing is snatched away.
3. **Ma before and after.** Where possible, delay entrances by `--duration-ma` (80ms) after the trigger. Delay the *next* action by the same interval after the entrance completes.
4. **Reduced motion.** All animations must respect `prefers-reduced-motion: reduce`. Fade only — no translate, no scale.

```css
@media (prefers-reduced-motion: reduce) {
  [class*="fude"], [class*="zanshin"] {
    animation-duration: 1ms !important;
    transition-duration: 1ms !important;
  }
}
```

5. **Never animate the ao-sumi ground itself.** Background color transitions on the body or root surface break the illusion of a washi scroll. The ground is permanent; marks appear on it.
